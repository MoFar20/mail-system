📧 THM Mail-System

About

THM Mail-System is a full-stack email management application developed as a university project for the Web Technologies course.

Features

  • 📥 Inbox Management - View and organize your emails
  • ✍️ Compose - Create and send new emails
  • ⭐ Star & Mark - Mark important emails
  • 📝 Drafts - Save emails for later
  • 🔍 Search - Find emails quickly
  • 🔒 JWT Authentication - Secure login

Technology Stack

Frontend: Angular 19, TypeScript
Backend: Spring Boot 4, Kotlin
Database: H2 (In-memory)
Security: JWT Authentication
